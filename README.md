## Raycast Scripts & Extensions

Local Raycast extensions for personal automation. See `docs/raycast-dev.md` for the development playbook.

### Extensions

- `browser-tab-switcher`: List tabs across Safari/Chrome/Brave/Edge and switch to a selected tab.
- `browser-tab-bulk-close`: Filter open tabs and bulk close matches across browsers (with confirm phrase).

### Quickstart

1) Import via Raycast → Create Extension → Use Local Folder (pick an extension folder)
2) In the extension folder:
   - `npm install`
   - `npx @raycast/api dev`
3) Run the command from Raycast. Approve Automation prompts for browsers.
